# グラフチャートv2

データセット（DS_MAIN）を元に，GRAPH CHARTを用いて，可視化する．

![グラフ](image/gra04.png)

パラメタ | 値
|----|----|
SOURCE | ユーザーID＋ユーザー名
TARGET | タスク名
METRIC | 回数
SOURCE CATEGORY | ユーザーID＋ユーザー名
TARGET CATEGORY | タスク名
GRAPH LAYOUT | FORCE
FILTER | ユーザー属性＝学習者（Learner）<br>コースID＝４　※評価実験用に限定<br>タスク名=NULL以外
